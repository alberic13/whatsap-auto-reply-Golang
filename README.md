# Auto Reply Bot (WhatsApp & Telegram)

This project is an auto-reply message bot integrated with a custom MySQL database and Gemini AI. The bot automatically responds to messages on both WhatsApp and Telegram.

## 🚀 Try It Out
* **Telegram**: [t.me/zalde_bot](https://t.me/zalde_bot)
* **WhatsApp**: `081381998561`

## 💬 Example Interaction
* **User**: `"halo"`
* **Bot**: `"can i help you ? "`

---

## ⚙️ How It Works
* **Keyword Trigger**: If a user sends a specific keyword, the bot checks your custom MySQL database and replies with the matching message.
* **Gemini AI Trigger**: If a user sends a message starting with `/ai`, the bot routes the query to Google Gemini AI to generate an intelligent response.
